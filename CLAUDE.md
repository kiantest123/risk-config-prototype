# risk-config-prototype

风控配置平台产品原型，Vite + React + TypeScript + Tailwind CSS。

## 部署

- **平台**：GitHub Pages
- **仓库**：https://github.com/kiantest123/risk-config-prototype
- **线上地址**：https://kiantest123.github.io/risk-config-prototype/
- **自动部署**：push 到 `main` 分支即自动触发构建和部署，无需手动操作
- **Workflow 文件**：`.github/workflows/deploy.yml`

## 本地开发

```bash
npm install
npm run dev       # 启动开发服务器 http://localhost:3000
npm run build     # 构建生产包到 dist/
```
